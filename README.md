# Tsinghua BID AI Screener

清华大学 BID 商业模式工坊 AI 招新筛选系统。

## 功能

- **候选人端**：在线完成面试问卷（基本信息 + AI 面试）
- **管理员端**：登录后台查看候选人数据、打分、筛选

## 本地运行

**前提**：Node.js

1. 安装依赖：`npm install`
2. 在 `.env.local` 中配置环境变量（参考 `.env.local` 文件中的说明）
3. 启动：`npm run dev`
